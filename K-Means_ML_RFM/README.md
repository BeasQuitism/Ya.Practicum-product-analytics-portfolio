### Поиск инсайтов, RFM-сегментация и кластеризация в сервисе доставки

**Задача:** Проанализировать пользовательский путь, оценить лояльность клиентов и выявить группы ресторанов для разработки персонализированных маркетинговых и бизнес-стратегий в сервисе доставки еды.

**Что сделал:**
- **Предобработка и EDA:** Провел очистку данных, работу с временными метками и исследовательский анализ трех датасетов (пользователи, события, заказы).
- **Анализ воронок:** Построил классическую воронку конверсии пользовательского пути (от авторизации до успешного заказа) и визуализировал долю пользователей на каждом шаге.
- **RFM-сегментация:** Рассчитал метрики Recency, Frequency и Monetary Value для оценки поведения клиентов. Разбил базу пользователей на группы по уровню лояльности и активности.
- **Кластеризация (Machine Learning):** Применил алгоритм K-Means для кластеризации ресторанов на основе их финансовых и операционных показателей (выручка, количество заказов, средний чек и др.).

**Результаты:**
- ✅ **Выявил узкие места в воронке:** Определил этапы с наибольшим оттоком пользователей (например, переход от просмотра меню к созданию заказа), что дает точку роста для UX-команды.
- ✅ **Создал основу для удержания (Retention):** Благодаря RFM-анализу выделил сегменты "уходящих" и "постоянных" клиентов, для которых можно настраивать триггерные рассылки и программы лояльности.
- ✅ **Оптимизировал работу с партнерами:** Разделил рестораны на кластеры (например, "лидеры по выручке", "стабильные середнячки", "нишевые"), что позволяет бизнесу разрабатывать индивидуальные условия сотрудничества и маркетинговые акции для каждой группы.

**Инструменты:** Python (pandas, matplotlib, seaborn), scikit-learn (K-Means), RFM-анализ, построение воронок конверсии.
