### Анализ юнит-экономики и системы метрик приложения (Procrastinate Pro+)

**Задача:** Проанализировать эффективность маркетинговых каналов, рассчитать ключевые продуктовые метрики и юнит-экономику для оптимизации бюджета на привлечение пользователей.

**Что сделал:**
- Рассчитал продуктовые метрики: DAU, MAU и Stickiness в разрезе каналов привлечения.
- Провёл когортный анализ: сгруппировал пользователей по дате и каналу привлечения для оценки их поведения во времени.
- Рассчитал unit-экономику: CAC (Customer Acquisition Cost), LTV (Lifetime Value) и ROI за первые 28 дней лайфтайма.
- Построил наглядные визуализации: heatmap для убыточных когорт, графики динамики ROI и Stickiness по месяцам.
- Выявил точки роста и убыточные зоны в монетизации продукта.

**Результаты:**
- ✅ Выявил каналы и конкретные когорты с отрицательным ROI (не окупились за 28 дней).
- ✅ Определил самые эффективные источники привлечения с лучшим показателем LTV/CAC.
- ✅ Сформулировал data-driven рекомендации для отдела маркетинга по перераспределению бюджета и отключению неэффективных рекламных каналов.

**Инструменты:** Python (pandas, matplotlib, seaborn), Jupyter Notebook

📁 **Файл:** `Проект_анализ юнит-экономики и системы метрик приложения.ipynb`
