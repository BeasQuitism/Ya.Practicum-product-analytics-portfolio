### Исследование рынка стартапов и инвестиционных трендов

**Задача:** Проанализировать датасет с данными о стартапах и раундах инвестиций, чтобы выявить закономерности привлечения финансирования, самые перспективные сегменты рынка и популярные типы инвестиций.

**Что сделал:**
- Загрузил и предобработал данные из базы стартапов (более 40 000 записей после очистки от аномалий и дубликатов).
- Разделил компании на 3 группы по срокам и количеству раундов финансирования: единичное, до года, более года.
- Сегментировал рынки на 3 категории: массовые (>120 компаний), средние (35–120), нишевые (<35).
- Проанализировал выбросы в каждом сегменте с помощью IQR-метода.
- Исследовал типы финансирования (venture, seed, angel, private equity и др.) по объёму привлечённых средств и популярности.
- Построил визуализации трендов инвестиций по годам и сегментам рынка.

**Результаты:**
- Определил, что венчурное финансирование (venture) генерирует наибольший объём возвратов и является лидером по популярности.
- Выявил наиболее перспективные и волатильные сегменты для инвестиций.
- Проанализировал долю общего объёма привлечённых средств по группам компаний и типам раундов.
- Подготовил аналитический отчёт с выводами и рекомендациями для потенциальных инвесторов.

**Инструменты:** Python (pandas, matplotlib, seaborn), Jupyter Notebook
