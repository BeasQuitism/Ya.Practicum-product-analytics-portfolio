## Проекты

### 1. 📊 A/B тестирование новой фичи в финтех-приложении

**Задача:** Оценить влияние нового онбординга на ключевые метрики приложения

**Что сделал:**
- Провёл полный цикл A/B тестирования на выборке 54,805 пользователей
- Проанализировал 4 метрики: ключевую (средняя сумма депозита), барьерную (конверсия в 1-й депозит) и 2 вспомогательные
- Применил статистические тесты: t-test, Mann-Whitney U test, bootstrap-анализ
- Построил воронку конверсии из 7 шагов

**Результаты:**
- ✅ Ключевая метрика выросла на **+2.45%** (статистически значимо, p < 0.0001)
- ✅ Конверсия из 1-го во 2-й депозит увеличилась на **+12.35 п.п.** (с 20.26% до 32.61%)
- ✅ Средняя сумма на платящего пользователя выросла на **+4.35%**

**Инструменты:** Python (pandas, scipy, statsmodels), SQL, matplotlib, seaborn

📁 **Файл:** `Проект_AB-анализ и поиск инсайтов_Финтех-приложение.ipynb`

## 4. 🔬 Проверка статистических гипотез и анализ A/B-теста интернет-магазина

**Задача:** Провести проверку статистических гипотез и оценить результаты A/B-тестирования новой версии сайта интернет-магазина

**Что сделал:**
- **Часть 1:** Проверил гипотезу о различии пользовательской активности между городами (Москва vs Санкт-Петербург) в приложении Яндекс.Книги
  - Проанализировал 8,540 пользователей (6,234 из Москвы, 2,306 из СПб)
  - Применил односторонний t-тест Уэлча для сравнения средних
  - Сформулировал гипотезы: H₀: μ_СПб ≤ μ_Москва vs H₁: μ_СПб > μ_Москва
  
- **Часть 2:** Провёл полный цикл анализа A/B-теста интернет-магазина BitMotion Kit
  - Оценил корректность распределения участников по группам (отклонение 1.56%)
  - Рассчитал необходимый размер выборки: 3,762 пользователя на группу (при базовой конверсии 30%, мощности 80%, достоверности 95%)
  - Проанализировал воронку конверсии за первые 7 дней лайфтайма
  - Применил Z-test для сравнения пропорций (конверсий в покупатели)

**Результаты:**
- **Часть 1:** p-value = 0.3436 > 0.05 → нет статистически значимых доказательств, что пользователи из СПб активнее
- **Часть 2:** 
  - Конверсия в контроле: 27.49%, в тесте: 29.27%
  - Фактический прирост: **+1.78 п.п.** (статистически значимо, p = 0.0203 < 0.05)
  - Ожидаемый эффект составлял 3%, фактический — 1.78% (меньше планового, но статистически значим)
  - Рекомендация: внедрить новую версию сайта, но учитывать, что эффект ниже ожидаемого

**Инструменты:** Python (pandas, scipy.stats, statsmodels), matplotlib, seaborn

📁 **Файл:** `Проект_Проверка гипотез в Python и результаты АБ-теста.ipynb`
