## Обо мне

Продуктовый аналитик с бэкграундом в финансах и опытом работы с данными. Специализируюсь на:
- A/B тестировании
- Когортном анализе
- Расчёте unit-экономики (CAC, LTV, ROI)
- Анализе продуктовых метрик (Retention, Conversion, DAU/MAU)

## Навыки

### Продуктовая аналитика
- Продуктовые метрики: Retention, LTV, CAC, ARPU, DAU/MAU, Conversion Rate
- A/B тестирование: дизайн эксперимента, расчёт sample size, статистическая значимость
- Когортный анализ и сегментация пользователей

- Unit-экономика

- **Python:** pandas, numpy, scipy, statsmodels, matplotlib, seaborn
- **SQL:** PostgreSQL, MS SQL
- **ClickHouse**, **Airflow**
- **BI-инструменты:** Power BI, DataLens
- **Статистика:** t-test, z-test, bootstrap, A/B тесты
