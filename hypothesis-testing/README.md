## Проверка статистических гипотез и анализ A/B-теста интернет-магазина

**Задача:** Провести проверку статистических гипотез и оценить результаты A/B-тестирования новой версии сайта интернет-магазина

**Что сделал:**

**Часть 1. Проверка гипотезы о пользовательской активности (Яндекс.Книги):**
- Проанализировал активность 8,540 пользователей (6,234 из Москвы, 2,306 из Санкт-Петербурга)
- Сформулировал гипотезы: H₀: μ_СПб ≤ μ_Москва vs H₁: μ_СПб > μ_Москва
- Применил односторонний t-тест Уэлча для сравнения средних значений
- Проверил условия для применения статистического теста (нормальность распределения, равенство дисперсий)

**Часть 2. Анализ A/B-теста интернет-магазина:**
- Проверил корректность распределения участников по группам (контрольная vs тестовая)
- Рассчитал необходимый размер выборки: 3,762 пользователя на группу (при базовой конверсии 30%, мощности 80%, уровне значимости 95%)
- Построил и проанализировал воронку конверсии за первые 7 дней лайфтайма
- Применил Z-test для сравнения пропорций (конверсий в покупатели) между группами
- Проверил статистическую значимость различий (p-value, α = 0.05)

**Результаты:**

**Часть 1:**
- ❌ p-value = 0.3436 > 0.05 → нет статистически значимых доказательств, что пользователи из СПб активнее пользователей из Москвы
- Нулевая гипотеза не отвергнута

**Часть 2:**
- ✅ Конверсия в контрольной группе: 27.49%, в тестовой: 29.27%
- ✅ Фактический прирост: **+1.78 п.п.** (статистически значимо, p = 0.0203 < 0.05)
- Ожидаемый эффект составлял 3%, фактический — 1.78% (меньше планового, но статистически значим)
- Сформулировал рекомендацию: внедрить новую версию сайта с учётом того, что эффект ниже ожидаемого

**Инструменты:** Python (pandas, scipy.stats, statsmodels), matplotlib, seaborn, Jupyter Notebook
